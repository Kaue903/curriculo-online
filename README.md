# 💻 Currículo Online - Kauê Rodrigues

Este é o meu portfólio pessoal e currículo online, desenvolvido para centralizar minhas informações profissionais, projetos e habilidades de forma interativa e responsiva.

## 📖 Sobre o Projeto
O projeto foi construído com foco em **Clean Code** e **UX/UI**, apresentando minha jornada como estudante de Desenvolvimento de Sistemas no SENAI e meu stack tecnológico. O site conta com modo escuro/claro e design totalmente responsivo.

### Seções do Site:
* **Início:** Apresentação com foco em Desenvolvimento Web.
* **Sobre Mim:** Detalhes sobre minha formação e objetivos profissionais.
* **Habilidades:** Visualização das tecnologias e ferramentas de trabalho.
* **Projetos:** Carrossel interativo com meus principais trabalhos.
* **Contato:** Formulário para mensagens e links sociais.
